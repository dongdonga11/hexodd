Part 1：本篇目标

  1. 本篇介绍窗体。涉及功能为：点击按钮后，弹出一个窗体
  2. 严格来说窗体不是控件，它是控件的载体

效果动图

Part 2：实现步骤

  1. 本项目中涉及到两个窗体

    • 窗体main作为主窗体，有一个按钮，点击后弹出窗体弹窗
    • 窗体弹窗，需要将其弹出方式选择是

  3. 窗体main中设置按钮对应的代码

Private Sub Command0_Click() 
    Dim frm   
    Dim ctrl 
    Dim frmName   
    frmName = "弹窗"    
    DoCmd.OpenForm (frmName)     
    Set frm = Forms(frmName)  
    frm.Caption = "说明"      
    Set ctrl = frm.Controls("Label4") 
    strSay = "这是一个测试弹窗功能的小程序"  
    ctrl.Caption = strSay
End Sub
