风格指南

  1. 组件名应尽量使用多单词组合，避免撞车关键词
    如班级学生组件，我将采用 klass-student 而避免使用 class 或者 student

  2. prop 使用参数校验

  3. 只应该拥有单个活跃实例的组件应该以 The 前缀命名，以示其唯一性。

  4. 和父组件紧密耦合的子组件应该以父组件名作为前缀命名。

  5. 模板中的组件名大小写
    使用小写+横线连接的形式如 my-component

  6. JS 、 VUE 文件命名遵循横线连接
